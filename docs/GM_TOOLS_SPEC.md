# GM Tools (General Manager) — Product Spec

The General Manager (GM) is our term for the League Commissioner. GM Tools are only visible and usable by the commissioner of a league.

---

## Pending Transactions

**Purpose:** Where the GM reviews and approves or declines roster changes for the whole league.

- **Pending roster adds** — Requests from owners to add a wrestler (e.g. from free agents).
- **Pending roster drops** — Requests from owners to drop a wrestler.
- **Pending trades** — Trade proposals between teams.

The GM can **accept** or **decline** each pending transaction. Non-GM users cannot manually add or remove wrestlers from their roster; they must submit an add and/or drop (and trades) that are then approved or declined by the GM here.

---

## Manage Rosters

**Purpose:** Where the GM can manually edit any team’s roster (add or remove wrestlers).

**Use cases:**

1. **Offline draft** — The league did an offline draft; the GM enters the results by manually adding wrestlers to each team’s roster.
2. **Draft errors** — Mistakes made during drafting (e.g. wrong wrestler assigned) that the GM needs to correct.
3. **Other corrections** — Any case where the GM needs to directly add or remove a wrestler from a team.

**Rules:**

- Only the GM can manually add or remove wrestlers from a roster.
- Non-GM users cannot use this page; they must use add/drop requests (and trades), which the GM handles on **Pending Transactions**.

---

## Notify League

**Purpose:** Where the GM sends messages to the entire league.

- **League feed** — Messages appear on the **League main page** (e.g. announcements, reminders).
- **Email** — The same message is sent by email to all league members.

This is the GM’s channel for announcements, draft reminders, deadline notices, etc.
